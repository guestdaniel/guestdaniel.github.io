LaTeX CV Template

Posted on August 27, 2017 in LaTeX

I use LaTeX for almost everything these days. LaTeX is a great way to create a high-quality CV, but I'm a big fan in particular because it's easy to include automated citations of all your papers and presentations via the biblatex package. I recently made some changes to a template I had found online to support APA style citations via the biblatex-apa package, and to automatically make my name bold in any citations I include in the CV.

I thought some of these changes might be useful to others, so I threw together an example CV using the template and put it up on my GitHub. The repository is here, but you can also download the whole thing as a .zip file here if you're not a git user! The whole thing should be fairly self explanatory if you're already familiar with LaTeX, but there's also a README and some comments in the .tex file in case you get lost. Send me any questions you have to guest121@umn.edu.
