LaTeX / KnitR Conference Poster Template

Posted on August 25, 2017 in LaTeX

Quickly into making my first conference poster with PowerPoint, I realized that I never wanted to make another poster in PowerPoint again. I often thought PowerPoint was frustrating to use and I found myself repeating the same things over and over again as I revised the poster (such as regenerating a figure in R to adjust something small and then importing and scaling the image in PowerPoint). Moreover, I prefer Linux over Windows, command line interface tools over GUIs, and open source software over proprietary software. This got me thinking about making a poster in LaTeX instead.

I did some research and found the beamerposter package as well as a great template and theme online and got to work. I'm fairly proud of my end result, even if the source code was not quite as pretty as the poster itself. My poster necessitated a lot of departures from the template I had started with. The biggest one was using a .Rnw source file and then using KnitR to turn it into a .tex file. This allowed me to include all the R code I had used to process my data and create my figures in the source file, and then to include the figures directly in the output document. To me, the three primary benfits of designing my poster this way were:

  1. I got access to all the power of LaTeX (e.g., great math typesetting)
  2. Changes to the analysis/figures were easy to implement on the fly
  3. The poster is readily reproducible

I've put together an example poster using an adapted version of the aforementioned template and theme for those who are interested in creating posters in LateX and who also use R. The files can be found in a GitHub repo here, and you can download them all as a .zip by clicking this link. README.md contains the a basic overview of the files and info on how to get started! Let me know if you have any questions by sending me an email.
